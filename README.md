# 빛과 바람이 머무는 집 - GitHub Pages

중개사와 매수 희망자에게 공유하기 위한 공개용 매물 소개 페이지입니다.

## GitHub Pages 게시

1. GitHub에서 공개 저장소를 만듭니다. 예: `sell-house`
2. 이 폴더 안의 파일과 `images` 폴더를 저장소 최상위에 업로드합니다.
3. 저장소의 **Settings → Pages**로 이동합니다.
4. **Deploy from a branch**, `main`, `/(root)`를 선택하고 저장합니다.
5. 잠시 후 `https://사용자명.github.io/sell-house/` 주소로 접속합니다.

PDF를 수정하면 `proposal.pdf`라는 같은 영문 파일명으로 교체해야 기존 링크가 유지됩니다.

PowerPoint 원본은 용량이 약 40MB이므로 공개용 폴더에서 제외합니다. GitHub 웹 브라우저 업로드는 파일당 25MiB까지만 지원합니다.
